接受企業培訓並執行各項工作任務，為將來準備要升任的管理職預先準備。

儲備幹部 (104薪資情報: 3.5萬～5.0萬/儲備幹部 )

負責企業或組織的營運策略規劃，並指導及協調各項相關工作事宜。

經營管理主管 (104薪資情報: 5.0萬～9.0萬/經營管理主管 )

依法於受有俸給之公務人員，及其他公營事業機關服務人員。或受機關委託，管理公共事務的人員。

公家機關相關人員 (104薪資情報: 3.5萬～5.0萬/公家機關相關人員 )

針對企業、團體或個人，從事商品或服務之國內外銷售、業務推廣、客戶開發等工作。

業務人員 (104薪資情報: 4.0萬～5.8萬/國內業務 ; 4.0萬～6.0萬/國外業務 )

在金融、投資相關企業、組織或部門中，從事有價證券受託買賣，如股票交易、期貨及選擇權等相關金融商品。

金融專業人員 (104薪資情報: 4.5萬～7.0萬/金融交易員 )

從事分析各項量化資料或資訊，提出與財務、稅務、投資、融資等相關之專業報告；督導並執行各項與資金有關之財務作業。

財務專業人員 (104薪資情報: 4.2萬～6.2萬/財務分析／財務人員 )

負責對當前及未來的經濟及財務性質進行研究分析，擬定企業投資計畫，亦負責記錄及資料處理公司的收入記錄如債券利息、股票紅利、抵押貸款之分期還款金額及不動產租金收入等資料。

金融投資人員 (104薪資情報: 4.7萬～7.2萬/金融研究員 )

於銀行、證券公司、保險公司、投信公司等機構內，從事資料蒐集、分析、撰寫研究報告之工作，提供投資決策考量之工作。

金融研究員 (104薪資情報: 4.7萬～7.2萬/金融研究員 )

從事企業或組織內部的財務業務之規劃、協調、指導、管制及考核的管理工作。

財務主管 (104薪資情報: 5.5萬～9.0萬/財務或會計主管 )

運用專業理財知識提供客戶理財規劃的建議及操作。
